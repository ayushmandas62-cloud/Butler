# Butler Desktop Architecture

## Vision

Butler Desktop is a native Linux desktop environment for Termux.

## Goals

- Native Termux
- No proot
- Stable startup
- Modular architecture
- Easy debugging
- Open source

## Version 0.1 Goal

Launch a stable XFCE desktop without crashes.
